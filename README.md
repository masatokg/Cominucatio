# Cominucatio

「htaccess」ファイルはサイトのトップフォルダに置いてファイルの先頭に「.」ピリオドをつけてください。
アクセスしているテーブルはユーザー情報を表すuserテーブルだけです。
userテーブルのカラム構成がわかるcreate table SQL文をつけてあります。
DBconnect.phpクラスに以下のようにデータベース接続の定義が書いてあります。
自分たちそれぞれのDB環境にあわせて動かしてみてください。
